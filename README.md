🎵 Spotify Music Player Clone

A responsive and visually appealing Spotify Music Player Clone built using core web technologies. This project aims to replicate the look and feel of Spotify while implementing essential music player functionalities.

🚧 Project Status: Currently in development

🚀 Features
🎧 Play and pause music
⏭️ Next and previous track controls
📊 Interactive progress bar
🔊 Volume control
📁 Playlist UI
💻 Responsive layout
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling
JavaScript (Vanilla JS) – Functionality

Open index.html in your browser
